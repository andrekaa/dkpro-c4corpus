C4Corpus

C4Corpus is a preprocessed CommonCrawl data set using DKPro C4CorpusTools including Creative Commons license detection, boilerplate removal, language detection, and near-duplicate removal.

See https://dkpro.github.io/dkpro-c4corpus/ for details.

File listing

warc.paths.gz contains a list of all warc.gz files available in this directory. The total size is 832 GB in 1,310 files.
